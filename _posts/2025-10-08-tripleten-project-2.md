# 📌 Project Overview
For this project, I was hired to help a client analyze the Manhattan vacation rental market. The goal was to provide guidance on which property types to invest in by uncovering key insights from Airbnb data.

### My analysis focused on answering two major questions: 
* Which neighborhoods and property sizes (i.e., number of bedrooms) are most attractive for vacation rentals?
* How much money did these listings generate?


## 🔎 Approach & Methodology

### Defining Attractiveness
To assess a listing’s attractiveness, I used the number_of_reviews_ltm column. Since reviews are typically left after a stay, they serve as a strong indicator of how often a property is rented.

### Data Cleaning Before building pivot tables 
I made sure the data was consistent and accurate: Cleaned the neighborhood column to fix inconsistent capitalization and remove trailing spaces. Stored cleaned values in a new column called neighborhood_clean. Created a copy of the raw dataset and documented all cleaning steps in a separate sheet for transparency.

### Pivot Table Analysis To answer the research questions
 Built pivot tables to analyze: The neighborhoods most attractive for vacation rentals. The property sizes (number of bedrooms) that are most popular. Whether different neighborhoods have different preferences for property sizes.

### 💡 Insights & Recommendations Based on my pivot table analysis, I was able to identify the top-performing neighborhoods and property types that generate the most engagement and income. These insights can guide strategic investment decisions for short-term rental properties in Manhattan.

## 📂 Project Deliverables

* Cleaned dataset with neighborhood_clean column
* Pivot tables highlighting rental activity and revenue by neighborhood & property size
* Dashboard with key findings (link above)
* Business recommendations for vacation rental investments

### ✨ This project showcases my ability to clean, analyze, and visualize real-world data using advanced spreadsheet techniques while providing actionable business insights.
